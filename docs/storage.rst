Storage
=======

This endpoint provides you the way to upload images on cyl.me project

/storage/image

+------------+------------+
| method     | GET        |
+------------+------------+
| Auth       | YES        |
+------------+------------+

Input vars

+-------------------+------------+---------------------------+
| Parameter         | Type       | Description               |
+===================+============+===========================+
| file              | string     | Image                     |
+-------------------+------------+---------------------------+

Response

+-------------------+------------+---------------------------+
| Parameter         | Type       | Description               |
+===================+============+===========================+
| id                | integer    | Image ID                  |
+-------------------+------------+---------------------------+
| preview           | string     | Relative path to image    |
+-------------------+------------+---------------------------+
