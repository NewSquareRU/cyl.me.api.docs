Categories
==========

Categories list
---------------

/categories/

+------------+------------+
| method     | GET        |
+------------+------------+


Response

+-------------------+------------+---------------------------+
| Parameter         | Type       | Description               |
+===================+============+===========================+
| categories        | array      |                           |
+-------------------+------------+---------------------------+
| Nested in categories                                       |
+-------------------+------------+---------------------------+
| category_id       | integer    | Category Id               |
+-------------------+------------+---------------------------+
| name              | string     | Category name             |
+-------------------+------------+---------------------------+

    {

                {

                    "name": "Немецкий язык"

            ]
        }
    }
