+------------+------------+
| method     | POST       |
+------------+------------+
| Auth       | NO         |

Input vars

+-------------------+------------+---------------------------+
| Parameter         | Type       | Description               |
+===================+============+===========================+
| email*            | string     | Email                     |
+-------------------+------------+---------------------------+
| name              | string     | User's name               |
+-------------------+------------+---------------------------+
| project           | string     | Project where email taken |
|                   |            | from(default: mabius)     |
+-------------------+------------+---------------------------+

Response

+-------------------+------------+---------------------------+
| Parameter         | Type       | Description               |
+===================+============+===========================+
| id                | integer    | Id                        |
+-------------------+------------+---------------------------+
| email             | string     | E-mail                    |
+-------------------+------------+---------------------------+
