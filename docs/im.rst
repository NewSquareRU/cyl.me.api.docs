Instant messaging
=================

Messages of user
----------------

/messages/{user_id}/

+------------+------------+
| method     | GET        |
+------------+------------+
| Auth       | YES        |
+------------+------------+


Parameters

+-------------------+------------+---------------------------+
| Parameter         | Type       | Description               |
+===================+============+===========================+
| user_id           | integer    | User's ID                 |
+-------------------+------------+---------------------------+


Response

+-------------------+------------+---------------------------+
| Parameter         | Type       | Description               |
+===================+============+===========================+
| messages          | array                                  |
+-------------------+------------+---------------------------+
| Nested in messages                                         |
+-------------------+------------+---------------------------+
| id                | integer    | Message Id                |
+-------------------+------------+---------------------------+
| user_id           | integer    | Use Id                    |
+-------------------+------------+---------------------------+
| message           | string     | Message                   |
+-------------------+------------+---------------------------+
| date              | timestamp  | Date                      |
+-------------------+------------+---------------------------+
| read              | 1|0        | Is message read           |
+-------------------+------------+---------------------------+
