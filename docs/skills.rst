Skills
======

Skills list
-----------

/skills/

+------------+------------+
| method     | GET        |
+------------+------------+
| Auth       | NO         |
+------------+------------+

Input vars

+-------------------+------------+---------------------------+
| Parameter         | Type       | Description               |
+===================+============+===========================+
| user_id           | integer    | Filter skills by user     |
+-------------------+------------+---------------------------+

Response

+-------------------+------------+---------------------------+
| Parameter         | Type       | Description               |
+===================+============+===========================+
| skills            | array      |                           |
+-------------------+------------+---------------------------+
| Nested in skills                                           |
+-------------------+------------+---------------------------+
| skill_id          | integer    | Skill Id                  |
+-------------------+------------+---------------------------+
| name              | string     | Skill name                |
+-------------------+------------+---------------------------+
