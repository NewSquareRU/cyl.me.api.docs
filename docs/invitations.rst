Invitations
===========

User's invitations
------------------

/invitations/

+------------+------------+
| method     | GET        |
+------------+------------+
| Auth       | YES        |
+------------+------------+

Input vars

+-------------------+------------+---------------------------+
| Parameter         | Type       | Description               |
+===================+============+===========================+
| user_id           | integer    | User's ID                 |
+-------------------+------------+---------------------------+


Response

+-------------------+------------+---------------------------+
| Parameter         | Type       | Description               |
+===================+============+===========================+
| invitations       | array                                  |
+-------------------+------------+---------------------------+
| Nested in invitations                                      |
+-------------------+------------+---------------------------+
| id                | integer    | Invitation Id             |
+-------------------+------------+---------------------------+
| user_id           | integer    | User Id                   |
+-------------------+------------+---------------------------+
| code              | string     | Invitation code           |
+-------------------+------------+---------------------------+
| used              | 1|0        | Was invitation used       |
+-------------------+------------+---------------------------+
