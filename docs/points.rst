https://api.cyl.me/v1/point/{point_id}/

+------------+------------+
| method     |    GET     |
+------------+------------+
| Авторизация|    нет     |
+------------+------------+

+------------+------------+------------------------+

|            |            |            |

+------------+------------+----------------------------+
| id         | integer    | Идентификатор точки        |
